# executeleads-demo

Branded landing page for the ExecuteLeads AI demo agent (a Tavus avatar of Angel).

Static only. **No secrets** Ś the Tavus conversation id is minted server-side and passed in the link as `?c=<id>`; no API key is ever in the browser.

Served at https://demo.executeleads.ai/ via GitHub Pages.
Source of truth: `Sales/ai_demo_agent/page/index.html` in the working repo.
